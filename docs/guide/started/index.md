# 开庭指引：从零开始使用 PyCourt

> 这是给第一次来到 Python Court 的你的快速导览。

1. **先把法庭搬进来（安装）**  
   阅读并完成：[安装与启动](install.md)

2. **让法庭认识你的疆域（配置）**  
   - 在项目根生成 `pycourt.yaml`，声明哪些文件享有治外法权；  
   - 在 `pyproject.toml` 中添加 `[tool.pycourt]`，划定文明领土。  
   参考：[配置指南](config.md)

3. **把审判变成日常开发流程（开发流程）**  
   学会何时使用匕首、军刀和节仗，把它们接到你的 AI 工作流上：  
   参考：[开发流程：成为 AI 指挥官](runtime.md)

4. **根据项目特点，精细调教豁免策略**  
   实际项目中的豁免策略示例：  
   参考：[豁免配置示例](exempt.md)

5. **最后：如果你愿意，一起来写法典 / 武器 / 故事**  
   - 想参与建设 PyCourt：看 [贡献指南](../community/contribute.md)  
   - 想借鉴更多经验：看 [常量与风格经验](../../exper/constants_style.md)
