
我们诚挚地邀请你加入**“联邦委员会”**，不仅仅作为用户，而是作为立法者和执法者，共同定义 AI 时代的编程文明。
1. 立法委员会

职责：制定规则，定义罪行

这是 PyCourt 最核心的权力机构。如果你发现了 AI 写出的新一种“恶心代码”，请向委员会提案。

提交法案 (Draft a Bill)：在 Issue 中使用 [New Law] 前缀，描述该罪行的特征。

获得冠名权：如果你的提案被采纳，这条法律将以你的名字命名（例如：Smith's Law），并被永久记录在 PyCourt 刑法典中。

晋升机制：成功通过 3 条法案的贡献者，将获邀成为**“终身立法委员”**，拥有对未来路线图的投票权。

2. 最高法院

职责：纠正冤案，维护正义

我们的审判算法（Linting Logic）可能并不完美，有时会判冤假错案（False Positives）。
提起上诉 (File an Appeal)：如果你认为 PyCourt 误判了你的代码，请提交 Issue 并带上 [Appeal] 标签。

司法解释：帮助我们优化检测逻辑，让审判更加公正。

3. 军械研发部
职责：锻造武器，跨平台化

我们需要精通 Rust / Go / Python CLI 的工匠。

目前的 Shell 脚本（匕首/军刀/节仗）只是原型。我们需要你加入研发部，将这些脚本重铸为跨平台的二进制神器，让 Windows 这里的“边疆地带”也能获得帝国的庇护。

4. 外交使团

职责：地域文化翻译，国际文化交流

我们在寻找不同文化背景下的“代码隐喻”。

跨文化交流：将“暗度陈仓”、“狐假虎威”等罪行翻译成你所在文化的幽默隐喻（如英语中的 "Trojan Horse" 或 "Potemkin Village"）。
